Subsystem: Thermal Control

Phase 1: Degradation Formulas

🔪 Blade — Linear Wear

wear = 0.01 × (1 + contam² × 5)
health -= wear

Quadratic amplification · 6× wear at max

🖨️ Nozzle — Weibull Hazard

excess = max(0, |temp| - 25)
dmg = (β/η)(excess/η)^(β-1)
β=2.5 η=50°C

Super-linear thermal fatigue

🔥 Heater — Exp Decay

cum_load += load
health = e^(-0.008 × cum_load)

Classic R(t) = e^(-λt)

📤 Output: StateReport

FUNCTIONAL >0.7 DEGRADED >0.3 CRITICAL >0.0 FAILED =0.0

📥 4 Input Drivers

temperature_stress · contamination
operational_load · maintenance_level

Go-Further: PINN

Physics-Informed Neural Network · Replace HeatingElements formula with learned model

🧠 Architecture

Input: cumulative_load (1)
FC(64)+Tanh → FC(64)+Tanh
→ FC(32)+Tanh → FC(1)+Sigmoid
Output: health ∈ [0, 1]

~5K params · Tanh for smooth autograd

📐 Dual Loss

L = L_data + λ·L_physics

L_data = MSE(pred, true)
L_physics = (dH/dx + λ·H)²

ODE: dH/dt = -λ·load·H

RMSE0.0039
